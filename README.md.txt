# 📊 Regressão Linear para Previsão de Faturamento

Projeto em Python aplicando regressão linear manual e com bibliotecas.

## 🚀 Tecnologias
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📈 O que o projeto faz
- Calcula regressão linear manual
- Compara com Statsmodels e Sklearn
- Avalia com R² e RMSE
- Gera gráficos
